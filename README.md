# Ontology Visualisation

## Introduction

Ontology visualisation web app based on NextJs/React/Redux-zero with internationalisation i18n support.

## Prerequisites

- [NodeJS](https://nodejs.org/en/download/)

## Installation

`npm install`

### Build

`npm run build`

### Run dev server

`npm run dev` # opens on http://localhost:3000

### Testing and Linting

```shell script
  npm run test
  npm run test:watch
```

```shell script
  npm run lint
```
